## 🛠️ 问题与解决 (Issues & Solutions)

1. **WooCommerce API 写入路径问题**
   - **问题**：部分环境直接调用 API 无法写入。
   - **解决**：API 路径中需要显式增加 `/index.php/` 才能正常写入数据。

2. **图片加速与格式兼容**
   - **方案**：使用 `weserv.nl` 免费图片加速服务。
   - **效果**：不仅加速了图片加载，还解决了 AVIF 格式图片无法写入 WooCommerce 的问题。
